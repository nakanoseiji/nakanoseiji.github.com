Play2.1 と Eclipse を Mac で

Jul 31st, 2013

久しぶりに環境を整えてみたので忘れないようにメモ。

それぞれのサイトから以下を持ってくる

  • eclipse-standard-kepler-R-macosx-cocoa-x86_64.tar.gz
  • pleiades_1.4.0.zip
  • play-2.1.2.zip

Eclipse

適当に解凍する。まだ起動はしない。 のちのち、以下のプラグインをインストール。

  • Java implementation of Git
  • m2e
  • Scala IDE

Pleiades

解凍し、plugins と features の中身をそれぞれ eclipse の対応するディレクトリの中に配置。

その後、eclipse を回答した先にある

に、

を追記。

一度起動する。 多分プラグインを追加したりした場合、ini ファイルと同じ場所にある eclipse を

で起動するものだと思われる。

Play

解凍して任意の場所に配置して、中にある play にパスが通った状態にする(お好み)。

プロジェクト作成

適当なディレクトリで、play コマンドを実行。

project_name ディレクトリが生成し、その中にファイルが一式揃った状態になる。

ちゃんとバージョン管理する。別にあとからでもよいですが。

作成したプロジェクトを Eclipse のプロジェクトにする。

Eclipse で開く

ファイルメニューのインポートから、”一般” の中にある “既存プロジェクトをワークスペースへ” を選択し、play で作成したプロジェクトのディレクトリを選択する。
